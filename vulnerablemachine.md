# Metasploitable2 vulnerable machine #

We will be using a vulnerable machine geared torwards being attacked with metasploit as it would fit being a victim of the framework as well. 

Download link: [http://sourceforge.net/projects/metasploitable/files/Metasploitable2/metasploitable-linux-2.0.0.zip/download]

How to download: 
- Extract the files from the .zip
- Go to VirtualBox and add a new machine, name the machine whatever you want, OS is Linux, then below that click Other Linux, 64 - bit
- Scroll to the bottom and add the .vmdk file in the Hard Disk section, click "Use an Existing Virtual Hard Disk File", then add it below
- Create VM, login and password are both msfadmin
